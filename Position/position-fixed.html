Position: static

Questa proprietà determina la posizione dell'elemento nel documento
NOTA BENE: LA PROPRIETÀ POSITION NON È INTESA COME STRUMENTO DI IMPAGINAZIONE

Per impaginare in maniera corretta si usa display

ma può essere utile per fare micro aggiustamenti o andare a sovrapporre coscenziosamente alcuni elementi.

position: static è la condizione di default, applicarlo di base non cambiarebbe il modo in cui l'elemento si comporta. Ma si può usare pe resettare il suo comportamento nel caso la sua position fosse stata applicata, per esempio tramite una media query

questo paragrafo hè posizionato con "position: static"

Il paragrafo sopra non cambia la sua posizione perché position: static è il valore di default di un elemento
